先用「兩個市場」理解 IS－LM

IS＝大家花錢的世界

政府支出、稅收、消費、投資，會影響整體支出。

看到政府花錢或加減稅，先想到 IS。

LM＝錢與利率的世界

央行、貨幣供給、升息、降息，會影響利率。

看到央行或利率，先想到 LM。

考題通常不是要你算，而是要你判斷：哪一條線移動、向左或向右、利率 r 與所得 Y 如何變化。

把它想成兩條路

IS：右下斜 → 利率低 → 投資多 → 所得高

LM：右上斜 → 所得高 → 用錢需求多 → 利率高

考前只先背這四句

政府多花錢

G↑ 或 T↓ → IS 右移 → Y↑、r↑

政府少花錢

G↓ 或 T↑ → IS 左移 → Y↓、r↓

央行放錢

M↑／降息 → LM 右移 → r↓、Y↑

央行收錢

M↓／升息 → LM 左移 → r↑、Y↓

口訣：政府看 IS，央行看 LM；放鬆往右，緊縮往左。
