# EMBA AI Learning Hub

## 7/22 總體經濟考試複習
115 管研 Lawrence｜20260718

這是一個可直接部署到 GitHub Pages 的純前端互動學習網站。

### 部署方式
1. 將 `index.html` 上傳到 GitHub repository 根目錄。
2. 到 Settings → Pages。
3. Source 選擇 `Deploy from a branch`。
4. Branch 選 `main`，資料夾選 `/root`。
5. 儲存後等待約 1–3 分鐘。

### 功能
- 小白教室
- IS–LM 互動圖
- AD–AS 情境模擬
- 匯率與固定／浮動匯率政策比較
- 翻卡複習
- 是非＋選擇題題庫
- 10題模擬考
- 錯題本
- 學習進度
